# Web_Projekt_Demo
Dieses Projekt ist eine kleine Web-Landingpage, die ich entwickelt habe, um meine Fähigkeiten in der Frontend-Entwicklung zu verbessern.
## Technologien
- HTML5
- CSS3 (Flexbox, Responsive Design)
- jQuery

## Funktionen
- Strukturierte und moderne Benutzeroberfläche
- Interaktive Elemente mit jQuery (FAQ, Filter, Scroll-Funktion)
- Responsive Layout für Desktop und Mobile
- Formular mit einfacher Validierung

## Hintergrund
Das Projekt entstand im Rahmen meiner Informatikausbildung EFZ. Dabei habe ich besonderen Wert auf eine saubere Struktur, einfache Benutzerführung und moderne Gestaltung gelegt.
